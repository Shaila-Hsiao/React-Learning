## 在 flaskServer 底下 run
```cmd=
python app.py
```


## 檔案介紹
### myModule
- connectDB.py 連接資料庫
- item.py 上傳 model 資料
- user.py user 登入、註冊
- convert_obj_three.py 將 obj model 轉成 js file
